# Panance
Python 3 Binance API wrapper built over Pandas and CCXT Libraries

* **Author**: Daniel J. Umpierrez
* **Version**: 0.1.0

# Requirements
 * python >= 3.6
 * pandas >= 23.0
 * ccxt

# Changelog
